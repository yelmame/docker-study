docker exec -it web1 bash

docker commit method we create image from container 
docker commit container-id nginx-new
save the image to file 
docker save-o nginx-new.tar nginx-new.latest
load the image 
docker load -i nginx-new.tar 
this will be created but no name (its called dangling image )
